# Eventual
Cada vez que se haga el pull, se tendran que ejcutar los siguientes comandos: </br>
cd front-end</br>
npm install</br>
cd ..</br>
cd back-end</br>
npm install</br>

Y así asegurar que vamos al día de los últimos paquetes. </br>
